♠ ルール・役一覧
アクション説明
チェック
ベットがない場合、チップを払わずにパスする。全員がチェックするとストリートが終了。
フォールド
ハンドを捨てて降りる。そのハンドへの参加権を失うが、以降の損失もない。
コール
直前のベット/レイズ額と同額を支払って続ける。
ベット
このストリートで最初にチップを賭ける。他のプレイヤーはコール・レイズ・フォールドを迫られる。
レイズ
直前のベットをさらに上乗せして賭ける。最低レイズ額はBBサイズ（または直前のレイズ幅）以上。
オールイン
所持チップを全て賭ける。コール額がスタックを上回る場合は自動的にオールイン。
ベット/レイズの宣言ルール
基本：宣言額 ＝ そのストリートでの最終到達額

「ベット 1000」→ 自分がそのストリートで合計 1000 投入する

「レイズ 5000」→ 自分がそのストリートで合計 5000 投入する（コール分も含む）

上乗せ額を宣言するのではなく、到達額を宣言します。

具体例（フロップ）
A
ベット 2000 → フロップ合計 2,000
B
コール → フロップ合計 2,000（Aと同額）
C
「レイズ、5000」と宣言
→ フロップ合計 5,000。追加で払うのは 3,000（5,000 − コール分 2,000）
最低レイズ額のルール
レイズの最低上乗せ幅は 直前のベット/レイズの上乗せ幅と同額以上。
例：BB=100 でAが 300 にレイズ（上乗せ幅 200）した場合、
次にレイズするBの最低到達額は 500（300 + 200）。
BがちょうどBが 500 でレイズした場合（上乗せ幅 200）、Cの最低到達額は 700（500 + 200）。
もしBが 700 にレイズしていたら（上乗せ幅 400）、Cの最低到達額は 1100（700 + 400）。
→ 直前のレイズ幅によって最低額は変動します。

ベッティング方式の種類
ブラインド方式 SB / BB

ディーラーボタンの左隣がスモールブラインド（SB）、その隣がビッグブラインド（BB＝SBの2倍）を強制的に払う。他の参加者はカード配布前に何も払わない。

  • 毎ハンドで払うのはSB・BBの2人だけ。ボタン移動と共に順番に回る
  • BBオプション：誰もレイズしなければ、BBの人は追加で払わずそのまま見られる特権がある
  • カジノ・トーナメントで最も広く使われる標準的な方式
アンティ方式 全員均等

ブラインドを使わず、毎ハンド参加者全員が同じ額をポットに投入してからゲームを開始する。特定の2人だけが払うブラインドとは異なり、全員が公平に負担する。

  • 全員が毎回同額払うため「タダで見られる」特権を持つ人がいない
  • みんなで少しずつ積み上げる感覚に近く、カジュアルなホームゲームで好まれやすい
  • プリフロップの最初のアクションはボタンの次の人からシンプルに開始（SB/BB位置を意識しなくてよい）
